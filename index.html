Project Gallery

Automata

Automata is a prototype of an automatic asphalt road damage detection system that uses transfer learning methods and the Tensorflow Object Detection library. This project has been contested in KMIPN IV and PKM 2022.

Automata Web

Website Automata as a complement to the detection system, namely as a website-based automatic mapping system that uses the GMaps API, JS and PHP.

Smart Trash IoT

Smart Trash IoT is a project from Diskominfo Madiun City as the development of the ICT field in the office as well as an internship assignment. Smart Trash IoT uses Infrared and Gas Sensors, Firebase RTDB as its database and ESP32 as its microcontroller programmed in C/C++ with Arduino IDE.

PPE Detection
